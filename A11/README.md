## Overview
This activity explores five unique access control devices used in both physical and digital environments. Each device is supported with evidence and a brief explanation.

---

## 1. Keycard Reader (RFID / Magnetic Stripe)
Evidence:
- Photo of apartment / building access card  

Explanation:
Keycard readers use RFID or magnetic stripe technology to authenticate users. When the card is scanned, the system verifies the credential and grants or denies access.

---

## 2. Biometric Authentication (Face ID / Fingerprint)
Evidence:
- Photo or screenshot of smartphone Face ID / fingerprint unlock  

Explanation:
Biometric systems use unique biological characteristics such as facial recognition or fingerprints to verify identity, providing high security.

---

## 3. PIN Keypad
Evidence:
- Photo of ATM keypad 

Explanation:
Users enter a personal identification number (PIN) to gain access. The system checks the entered code against stored data.

---

## 4. Password Authentication
Evidence:
- Screenshot of login page  

Explanation:
Password authentication requires users to input a secret string. It is one of the most common forms of digital access control.

---

## 5. Two-Factor Authentication (2FA)
Evidence:
- Screenshot of OTP code 

Explanation:
2FA adds an extra layer of security by requiring a second verification step, such as a one-time password (OTP), in addition to a password.

---

## How Access Control Works
1. Identification – User provides credentials  
2. Authentication – System verifies identity  
3. Authorization – Access is granted or denied  
4. Logging – Activity is recorded  

---

## Conclusion
Access control devices are essential for protecting both physical and digital systems. Modern security increasingly combines multiple methods such as biometrics and 2FA to improve protection.
