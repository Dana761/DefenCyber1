# A12 – Offline Security Tools


---

## Overview
This project demonstrates five unique **offline security tools** used to protect systems without relying on internet connectivity. These tools are commonly used in secure or air-gapped environments.

---

## Tools & Evidence

### 1. ClamAV (Offline Antivirus)
- **Evidence:** Terminal output of file scan  
- **Description:** Open-source antivirus that scans files using locally stored virus definitions.

---

### 2. XProtect (macOS Built-in Antivirus)
- **Evidence:** System Information showing XProtect components  
- **Description:** macOS built-in antivirus that detects malware using offline signatures.

---

### 3. ZIP Encryption (macOS Built-in)
- **Evidence:** Password-protected ZIP file  
- **Description:** Secures files locally using password-based encryption.

---

### 4. macOS Firewall
- **Evidence:** Firewall enabled in system settings  
- **Description:** Blocks unauthorized incoming connections without internet dependency.

---

### 5. Keychain Access (Password Manager)
- **Evidence:** Screenshot of stored credentials  
- **Description:** Stores passwords securely and works offline for authentication.
