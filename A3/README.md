# A3 – Security Concepts Used in My House

##  Description
In my room, access is controlled using an **electronic access control system**. The door can only be unlocked using a **registered access card**. To enter, the user must tap the card on a **black circular RFID/NFC reader** installed above the door handle.

This system replaces traditional keys and provides better security and monitoring.

---

##  Security Concepts Identified

### 1. Authentication
The system verifies the identity of the user through a **registered access card**.  
Only cards that are stored in the system database are accepted.


---

### 2. Authorization
Even if a card is detected, access is only granted if the card has **permission to unlock the door**.

➡️ Not all cards can open the door — only authorized ones.

---

### 3. Access Control
The door uses an **electronic lock system** that restricts entry.  
Without tapping a valid card, the door remains locked.

➡️ Prevents unauthorized physical access

---

### 4. Identification Technology (RFID/NFC)
The black circular device is an **RFID/NFC reader** that communicates with the access card wirelessly.

➡️ Enables contactless authentication

---

### 5. Physical Security Integration
This system combines:
- Physical barrier (door + lock)
- Digital authentication (card system)

➡️ Example of **cyber + physical security integration**

---


## 🧠 Reflection
Using an access card system is more secure than traditional keys because:
- Cards can be easily **disabled or replaced**
- No risk of copying physical keys
- Provides **controlled and trackable access**

However, potential risks include:
- Lost or stolen cards
- System malfunction (e.g., power failure)

---

## ✅ Conclusion
The access card system demonstrates multiple cybersecurity concepts in a real-world home environment, including **authentication, authorization, and access control**, combined with modern **RFID/NFC technology**.
