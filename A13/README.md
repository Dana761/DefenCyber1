# A13 – Discover 5 Unique Online Security Tools

## Introduction
Online security tools play a critical role in protecting modern cloud-based systems, applications, and users. These tools operate in real time over the internet, enabling continuous monitoring, threat detection, and automated responses. This portfolio presents five unique online security tools along with supporting evidence and explanations.

---

## 1. Cloudflare Web Application Firewall (WAF)

**Type:** Online Security Tool  

**Evidence:**  
- Screenshot of Cloudflare dashboard showing security features such as "Protect apps and users"  
- Demonstrates real-time traffic filtering and protection  

**Description:**  
Cloudflare WAF is a cloud-based security service that protects websites and applications by filtering malicious traffic and blocking attacks such as DDoS and SQL injection in real time.  

**Additional Notes:**  
Cloudflare operates as a reverse proxy, meaning all traffic passes through its network before reaching the website, allowing it to detect and block threats efficiently.

---

## 2. Intruder

**Type:** Online Security Tool  

**Evidence:**  
- Screenshot of Intruder dashboard showing vulnerabilities and threat levels (e.g., Critical)  
- Demonstrates continuous vulnerability scanning  

**Description:**  
Intruder is a cloud-based vulnerability management platform that continuously scans systems for security weaknesses, misconfigurations, and known vulnerabilities.  

**Additional Notes:**  
It helps organizations reduce risk by identifying vulnerabilities before attackers can exploit them.

---

## 3. Check Point SASE

**Type:** Online Security Tool  

**Evidence:**  
- Screenshot from official Check Point SASE documentation showing architecture and security use cases  
- Full PDF document included in repository  

**Description:**  
Check Point SASE is a cloud-delivered security solution that provides secure access, threat prevention, and data protection using a zero-trust architecture.  

**Additional Notes:**  
It integrates browser-level protection, cloud inspection, and identity-based access control to secure distributed environments.

---

## 4. SentinelOne Singularity Cloud Security

**Type:** Online Security Tool  

**Evidence:**  
- Screenshot of SentinelOne dashboard displaying critical alerts and detected threats (e.g., ransomware)  
- Shows real-time monitoring and security analytics  

**Description:**  
SentinelOne is an AI-driven cloud security platform that detects, analyzes, and automatically responds to cyber threats across systems and cloud environments.  

**Additional Notes:**  
It uses behavioral AI to identify abnormal activities and mitigate threats without human intervention.

---

## 5. Microsoft Defender for Cloud

**Type:** Online Security Tool  

**Evidence:**  
- Screenshot of Microsoft Defender dashboard showing threat intelligence and attacker profiles  
- Demonstrates real-time threat monitoring and analysis  

**Description:**  
Microsoft Defender for Cloud is a cloud-based security platform that provides threat detection, vulnerability management, and centralized security monitoring across cloud services.  

**Additional Notes:**  
It leverages Microsoft threat intelligence to track attackers and provide actionable security insights.

---

## Conclusion
These five online security tools highlight the importance of cloud-based cybersecurity solutions in modern environments. They provide real-time monitoring, vulnerability detection, and advanced threat protection using technologies such as AI and zero-trust architecture. This demonstrates a strong understanding of current cybersecurity tools and practices.
