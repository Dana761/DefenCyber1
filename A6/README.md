# A6 – Offline Cryptographic Implementation (macOS Encrypted Disk Image)

This folder contains my evidence for Activity A6, where I explored offline cryptography used to protect data stored locally, without any internet connection.

## Evidence Included
- **apfs_encrypted_volume.png** — Screenshot of macOS Disk Utility showing an APFS encrypted volume. This demonstrates full‑disk encryption used to secure data at rest.
- **disk_utility_encryption_options.png** — Screenshot showing AES‑128 and AES‑256 encryption options while creating an encrypted disk image.
- **encrypted_dmg_success.png** — Screenshot confirming successful creation of an encrypted .dmg file using macOS Disk Utility.

## Summary
macOS’s Disk Utility provides offline cryptographic protection using AES encryption to secure disk images and full APFS volumes. These mechanisms encrypt data locally on the device, ensuring confidentiality and integrity even when the system is offline.
