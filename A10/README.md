# A10 – Discover Privacy Technique Used Offline

## Overview
This task demonstrates an offline privacy technique that protects user data by ensuring information is stored locally on a device without relying on internet connectivity or cloud-based services.

---

## Privacy Technique: Offline Data Storage

Offline data storage is a privacy technique where data is kept entirely on a local device rather than being transmitted or synchronised with external servers. This approach reduces exposure to cyber threats such as data breaches, unauthorised access, and third-party tracking.

---

## Implementation
In this example, a built-in macOS application (TextEdit) is used to create and store a document locally.

- The file is saved directly on the device  
- No cloud synchronisation (e.g., iCloud) is used  
- Data remains fully under user control

---

## Why This Matters
Storing data locally improves privacy because:

- It prevents third-party access to personal data  
- It reduces the risk of cloud-based data breaches  
- It eliminates unnecessary data transmission over the internet  
- It ensures full user ownership and control of data  

---

## Conclusion
This activity demonstrates that offline data storage is an effective privacy technique. By keeping data on the device and avoiding cloud services, users can significantly reduce privacy risks and maintain better control over their information.
